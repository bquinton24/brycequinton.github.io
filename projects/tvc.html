Thrust Vector Control

Overview

Project to design a thrust vector control to use with a class I or J solid rocket motor.

Goals

  • Design and 3D print the gimbling motor tube to fit in a 4 inch diamter body tube.
  • Program the thrust vectoring system and simulate the C++ file through MathWork's Simulink.
  • Conduct static fire testing on the system.

Key Accomplishments

  • Data Automation: I look to have this project to be the main focus as Mercury V gets closer to completion.
  • Motor Research: I intend to use the I59WN due to it's long burn time.
